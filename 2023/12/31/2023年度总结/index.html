2023年度总结

发表于2023-12-31|更新于2024-12-28|杂谈
|阅读量:

黑夜给了我黑色的眼睛，

我却用它寻找光明。

—— 顾城《一代人》

2023 年即将过去，元旦前夕，我坐在电脑前回顾着这充满波折的一年。

高考

考点的大门口处排着几条蜿蜒曲折的队，从教学楼拐到了另一侧的教学楼，最后又排到了存真楼，像一条条随风飘拂的发丝。我正捧着一沓笔记排在队中，当时的心情……似乎无法用一个词准确概括，即为人生大考感到紧张，又因为之前的无数次模拟赛而已经麻木，也有一丝丝终将结束的欢喜。

考完数学走出考场的同时，雨点纷纷飘落，狂风又把雨水送到连廊底下，打在依靠连廊避雨的考生身上，也打在我的身上，衬托着我当时的心情。

考试结束的当天晚上，学校就打开大门让家长把学生都接走了，这是当时我在教学楼拍下的一幕：

存真楼下停着成片的汽车，天色已晚，存真楼的灯光映在楼檐上。之前学业繁忙，导致我已经很久没有仔细寻找四周的美，那天晚上在暮色中泛着微光的存真楼显得格外夺目。

我不禁想起在存真楼学习竞赛的那些时光，如今高中生活已经过去，新的篇章即将翻开，新的机遇和挑战也即将来临。

竞赛不是火，却能点亮一生。

—— 贴于存真楼 11 楼信息竞赛教室外墙

暑假

高校招生会前一天晚上，高考成绩出炉了，出乎我意料的是，最先知道成绩的途径不是从网站上主动查，而是在更早的时候就通过短信发到我手机上了：

数学物理各有挂分，出乎意料的是化学赋的这么低（或者是我考的太低了），总之副科也是要好好学滴。

查了查一分一档，这个档次确实很尴尬，按照去年的档次，想上 985 计算机很困难了，于是打算第二天咨询转专业。

第二天北校高招会，我们把中下流的 985 挨个问了一遍，中流 985 给出的结果都是“可以冲一冲”，可以冲一冲是什么意思？就是委婉地拒绝你辣。

于是乎，末流 985 和 211 也都问了一些。特意问了转专业、保研等问题。

很快，报志愿的日子过去了，在其他同学陆续收到了心仪大学的录取信息期间，我也得到了自己的录取信息，由于志愿查询的官网不给力，这个信息是从主动向吉大招生打电话确认的。

再之后，投档线也出了：

这 919KB，承载着多少高三生的欢喜与哀愁。

最终我被录取到「吉林大学-通信工程」，其投档线正好也是 628，卡线了属于是。

值得一提的是，我填报的「北京交通大学-理科试验班(数学与统计)」投档线是 629，也就是说，如果我多考一分或者少考一分，结局都会大不相同。不过现在看来，在吉大能开学进唐敖庆计算机挺不错的，而且不用整转专业那乱七八糟的事儿了。

北国风光 —— JLU

北国风光，千里冰封，万里雪飘。

—— 毛泽东《沁园春·雪》

吉林夏日风光：

入学后和通信工程的同学参加军训，通信学院人多到南湖校区装不下在南岭还有一些高年级的。

吉大校区真多，而且南湖南岭南区都是南开头的，刚开始很容易搞混。

放一张军训期间到南区进行急救训练的照片：

军训快要结束的时候我也完成了唐敖庆计算机的面试，得到了结果，搬到了南区旁边的吉大大学城。

吉大中心校区东西跨度很大，从大学城到校园东侧的三教步行要半小时，骑自行车可以大大减少通勤时间。

感谢迎新会上学长的指点和帮扶，有这么多热心的学长请教真不错，还要感谢邵老板向专教捐赠的打印设备和 A4 纸。

唐匡班的专教真不错，再也不用担心自习找不到地方了。

我在之后的班委选拔中成功当选班长，感谢同学们的支持~

放几张中心校区的雪景：

忙碌的竞赛月

来到中心校区之后，我就开始联系 ACM 教练，寻找队友准备参加 XCPC 竞赛，最终我和 ws 以及 qyb 组成了队伍。

进入到 10 月下旬，我们队连续迎来了三场 XCPC 比赛。

出师不利 —— CCPC 桂林站

由于这是组队后的第一场比赛，还没有磨合好，于是桂林站寄了。

飞往桂林：

桂林之景：

酒店后面就是凸起的峰林，不愧是喀斯特地貌。

紧贴着的两栋住宅楼。

比赛场地在桂林电子科技大学内，桂电也很大，在校园里能看到周围的峰林，桂林山水甲天下。

比赛场地：

发的参赛物资：

但是比赛过程吗，就一言难尽……

比赛没开始的时候就发现谁都没有带词典和板子。

开题之后，发现 M 就是二分答案加前缀和，在榜上一看发现已经有很多人过，于是开写。但是退役太久手生硬是写了半个多小时（buff 1）。

之后跟榜陆续过了 G、B 和 I。

G 的题意就是模拟编辑器光标。

B 是一道比较小清新的题，只有一种操作，解法是贪心。

I 我没看，赛后看代码挺长的，此题有三发罚时（buff 2）。

然后跟榜写 K 题，思路很简单，拿一个 set 存迭代完前 i 个下标后的所有序列，然后每次迭代暴力求出所有新组合成的序列然后存到 set 中即可。但是始终过不去，就这样找了很久的 bug，总算找到原因得以通过，原来竟是没有开两个 set 滚动存状态，只开了一个 set 那必过不了啊。于是这题又耽误了不少时间（buff 3）。

最后，银牌题 C 题是都不擅长的数论（buff 4），其他题也不太可做。

铜牌第 6，寄了。

飞离桂林：

从飞机上俯瞰桂林，景色真的很美。

高手云集 —— ICPC 南京站

相比于桂林，南京和沈阳的街景其实就和北京和长春的有些类似了，也没有像桂林那样瑰丽的喀斯特地貌，就没感到太多新奇。

到达南京当晚受 yzh 之邀去小厨娘淮阴菜吃了顿当地特色的菜肴，好评。

比赛场地在南京航空航天大学内，南京站的场馆条件我感觉是今年去过的 3 站中最好的了。尤其是桌椅，桌子很大，能轻松把所有比赛用品放下，甚至还有空间放午饭，中间有隔板将两个队伍隔开。

比赛场地：

好消息是我带上了词典和自己打印的板子，不过这场比赛居然发中文题面了（什么时候普及到其他赛站啊）太贴心了。

不得不提南京站的特色——袋鼠，热身赛的赛题全是袋鼠，每个队还会发 3 只袋鼠吉祥物：

I 题，签到题。

G 题，贪心+dp，对于固定的选法，免费次数一定是给售价高的宝石用更优，按费用排序后枚举分界点，售价高的部分用贪心，售价低的部分用 dp。

C 题我没看，好像是数学题。

F 题唯一的要求“重排后的生成序列和原来的生成序列相等”可以等价为若干个限制要求，每个限制要求为“第 i 个操作必须排在第 j 个操作的前面”，根据这些限制要求建图，看一看有没有除了原排列以外的拓扑序即可。

A 题，袋鼠题，我没看，好像是搜索+模拟。

L 题，有重量为 1 和 2 的两种货物，分类讨论贪心。思路很简单，但是又很多细节，题目给的是若干组货物，每组货物的一样的，但是有若干个，直接一个一个货物贪心肯定是会超时的，必须一组一组考虑，于是就有了若干细节。

最后 M 题写不动了，开始扫雷。

值得一提的是，由于长春大雪，回去的飞机票取消了，只得订高铁。由于大雪高铁减速，从上午坐到将近午夜，终于到长春以后还打不到车，最终在凌晨的时候才叫到网约车。

收官之战 —— ICPC 沈阳站

最近的一站，来回路程短多了，高铁上打个盹就过去了。

订的旅馆好破啊，早知道订个好点的了，反之能报销。不过旅馆到比赛场地很近。

不知道四周有什么好吃的饭馆，就订了汉堡薯条，结果吃了几天都快吃吐了。比赛当天发的午饭也是汉堡，真吃不下了，当天午饭就没吃。

C 题，签到题，阅读理解题，读懂题意后打个表就切了。

J 题，AliceBob 一眼博弈，感觉不是我擅长的类型，没看。

K 题，数据结构题。

E 题，狼羊题，数据范围很小，一眼 dp，但是离奇的是，此题死活过不了，期间我无数次地打印代码，和队友一起检查，就是找不到任何问题。最后没有办法让队友重构了一遍，结果一发过了，更离奇的是，队友说除了把 int l, r, que[MAXn] 换成 queue<int> 之外其他部分和我的代码一样。但是这个队列我也检查过了是不会爆的啊，为什么呢？

B 题，没仔细看。

最后 M 题找到规律但是没来及写，或者说是状态太差写不动了。要是前面别卡那么久，然后结尾的时候能有更多时间把 M 写出来就有可能金了。

离开沈阳：
